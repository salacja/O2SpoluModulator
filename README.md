# O2 Spolu Modelator – Landing Page

High-fidelity landing page pro nákup balíčku O2 Spolu. Optimalizováno pro konverzi a srozumitelnost.

## UX řešení

- **Single-page landing** – vše na jedné stránce, minimální friction
- **Value-first Hero** – úspora až 450 Kč hned v nadpise
- **„Co je O2 Spolu“** – inline vysvětlení (řeší nepochopení služby)
- **Doporučené balíčky** – 3 karty s cenou a úsporou
- **Checkout drawer** – slide-in panel místo přechodu na novou stránku

## Flow

1. Hero → scroll k balíčkům nebo klik na CTA
2. Explainer „Co je O2 Spolu“ (3 kroky)
3. Výběr balíčku → otevření draweru
4. Potvrzení → Shrnutí → Objednat

## Spuštění

```bash
npm install
npm run dev
```

## Design

- O2 barvy (modrá #0066CC)
- DM Sans typografie
- Mobile-first, responzivní
- Úspory zvýrazněny zeleně
